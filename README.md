# 📊 Data Handling Projects
Starting point to strengthen my foundation in **Data Preprocessing & Transformation** for AI/ML workflows.

---

## 🔎 About
This repository is dedicated to **data handling techniques** — the backbone of any AI system.  
It covers cleaning, integration, transformation, and exploratory analysis, ensuring raw datasets are turned into structured, reliable inputs for algorithms.

---

## 📌 Project List
1. **Data Cleaning with Pandas** – Handling missing values, duplicates, and inconsistencies.  
2. **Data Integration** – Merging multiple datasets and resolving schema conflicts.  
3. **Data Transformation** – Encoding categorical variables and applying feature scaling.  
4. **Exploratory Data Analysis (EDA)** – Visualizing distributions, correlations, and trends.  
5. **Outlier Detection** – Identifying anomalies using statistical and ML methods.  
6. **Feature Engineering** – Creating new features to improve model performance.  
7. **Dimensionality Reduction** – Applying PCA and other techniques for efficiency.
